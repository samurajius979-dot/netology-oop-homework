# Домашнее задание «ООП: наследование, инкапсуляция и полиморфизм»

Готовое решение практического задания Netology по теме ООП.

Репозиторий для сдачи:
[https://github.com/samurajius979-dot/netology-oop-homework](https://github.com/samurajius979-dot/netology-oop-homework)

## Что реализовано

- классы `Student`, `Mentor`, `Lecturer`, `Reviewer`;
- наследование `Lecturer` и `Reviewer` от `Mentor`;
- метод `Reviewer.rate_homework()` для выставления оценок студентам;
- метод `Student.rate_lecture()` для выставления оценок лекторам;
- хранение оценок в словарях по курсам;
- магические методы `__str__`, `__eq__`, `__lt__`;
- сравнение студентов и лекторов через операторы `>`, `<`, `==`;
- функции подсчета средней оценки по конкретному курсу;
- демонстрация работы на двух студентах, двух лекторах и двух проверяющих;
- встроенные `assert`-проверки ключевых требований.

## Запуск

```bash
python main.py
```
