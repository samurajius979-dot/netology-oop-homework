# Домашнее задание «ООП: наследование, инкапсуляция и полиморфизм»

Решение практического задания Netology по теме ООП.

В проекте реализованы:

- классы `Student`, `Mentor`, `Lecturer`, `Reviewer`;
- наследование `Lecturer` и `Reviewer` от `Mentor`;
- выставление оценок студентам проверяющими;
- выставление оценок лекторам студентами;
- магические методы `__str__`, `__lt__`, `__le__`, `__eq__`;
- функции подсчета средней оценки по курсу для студентов и лекторов;
- демонстрация работы всех методов на двух экземплярах каждого класса.

## Запуск

```bash
python main.py
```
